# Ludo Move Counter - React

## Description
This is a simple Ludo move counter game built using React. It tracks the number of moves for each player using buttons.

## Features
- Track moves for Blue, Red, Green, Yellow players
- Built using React useState
- Interactive UI
- State updates using spread operator

## Tech Used
- React.js
- JavaScript
- HTML
- CSS

- Event handling in React
- Updating object state
- Spread operator
